Awesome to do list

El primer proyecto con el que la mayoría nos enfrentamos, una lista de tareas manipulable, con estilo sencillo y elegante.

Descripción del proyecto

Este proyecto es una aplicación web simple que permite a los usuarios gestionar una lista de tareas. Los usuarios pueden agregar nuevas tareas, marcarlas como completadas y eliminarlas de la lista. La interfaz es intuitiva y fácil de usar, con un diseño limpio y moderno.

El proyecto está construido utilizando HTML para la estructura, CSS para el estilo y JavaScript para la funcionalidad. Se han implementado buenas prácticas de codificación y se ha prestado atención a la experiencia del usuario.
